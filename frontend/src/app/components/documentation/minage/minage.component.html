Minage

Le minage est le processus par lequel les utilisateurs de Bitcoin résolvent les puzzles mathématiques complexes pour confirmer les transactions et les ajouter à la blockchain. La personne qui résout le puzzle est récompensée avec des Bitcoins. Aujourd'hui, les mineurs récoltent 6.25 BTC par bloc miné et ce chiffre est divisé par 2 tous les 4 ans, lors d'un événement connu sous le nom de "Halving". Le minage est l'un des principaux mécanismes qui maintiennent Bitcoin sécurisé et protégé contre les fraudes.
